# Seattle Waste Dumpings

## Project Description
This project includes a thematic chloropleth map which displays the amount of illegal waste dumpings per neighborhood in Seattle. It also containes a legend and bar graph to help the user understand the map. Including a reset button feature which makes you go back to where you started on the map in case you get lost.
